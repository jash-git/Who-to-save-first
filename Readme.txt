我跟你爸要救誰？　本土劇給「標準答案」網蓋400樓讚：史上首位答對的男人 [先救人還是先救火][Who to save first]

資料來源: https://star.ettoday.net/news/1285096
https://www.youtube.com/watch?v=Yb5QNdDtrdQ


男女朋友交往時，不少男生都會被問到，「如果我跟你媽同時掉到海裡，你會先救誰？」無論選擇哪一邊都不是最好的答案，堪稱是男生們最頭痛的問題冠軍。本土劇也把這個問題納入劇情，只是將問題換成「我和你爸被困在火災裡，要先救誰」，男主角超冷靜又實際的回答讓網友全跪，笑讚：「史上第一個答對的男人！」

本土劇《兄弟姐妹》近日被挖出其中一個片段，女主角賴暄妮激動地拉住男主角秦楊的手，問說：「萬一我們家發生火災，我跟爸爸都在屋裡逃不出去，你會先救爸爸還是先救我？」此時秦楊一臉嚴肅地轉過身去，冷靜地回答：「我會先救火！」
